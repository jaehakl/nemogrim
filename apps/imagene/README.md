Imagene는 Stable Diffusion 기반의 AI 이미지 생성 및 관리 시스템입니다. 연구자와 아티스트가 대량의 이미지를 효율적으로 생성하고 체계적으로 관리할 수 있도록 돕는 전문 도구입니다.

---

## 🎯 주요 기능

### AI 이미지 생성
Imagene는 **Stable Diffusion XL** 모델을 사용하여 고품질 이미지를 생성합니다. 사용자는 **Seed, Steps, CFG** 등 다양한 파라미터를 설정하거나, 범위 내에서 무작위로 생성할 수 있습니다. **자동 반복 생성** 기능을 통해 대량의 데이터셋을 손쉽게 구축할 수 있으며, **프롬프트 생성기**와 **키워드 관리 시스템**을 활용해 효과적인 프롬프트를 만들 수 있습니다.

### 이미지 관리 및 분석
생성된 이미지는 그리드 형태의 갤러리에서 한눈에 볼 수 있으며, **키워드 검색**과 **그룹화** 기능을 통해 효율적으로 관리할 수 있습니다. **통계 분석** 기능을 이용해 생성 파라미터, 프롬프트 길이, 키워드 빈도 등을 분석하고, **상관관계 분석**으로 최적의 생성 조건을 찾을 수 있습니다.

## 🛠️ 기술 스택

### 프론트엔드
- **React 18**: UI 개발
- **RSuite**: 컴포넌트 라이브러리
- **Recharts**: 데이터 시각화
- **Context API**: 전역 상태 관리

### 백엔드
- **FastAPI**: 고성능 API 서버
- **SQLAlchemy 2.0**: 데이터베이스 ORM
- **PostgreSQL + pgvector**: 벡터 임베딩 지원 데이터베이스
- **PyTorch**: 딥러닝 프레임워크
- **Sentence Transformers**: 텍스트 임베딩

## 🚀 설치 및 실행

### 사전 요구사항
- Python 3.11+
- Node.js 18+
- CUDA 지원 GPU
- PostgreSQL (pgvector 확장 필요)

### 백엔드 실행
1. `cd apps/imagene/api`
2. `poetry install`로 의존성 설치
3. `.env.example`을 복사해 `.env` 파일 생성 후 설정
4. `poetry run uvicorn app.main:app --reload`로 서버 실행

### 프론트엔드 실행
1. `cd apps/imagene/ui`
2. `npm install`로 의존성 설치
3. `.env.local` 파일에 `VITE_API_BASE_URL=http://localhost:8000` 설정
4. `npm run dev`로 개발 서버 실행

## 📂 프로젝트 구조

Imagene는 **백엔드(api)**와 **프론트엔드(ui)**로 구성된 모놀리식 아키텍처를 가집니다.

- `apps/imagene/api`: FastAPI 기반의 백엔드 코드가 위치합니다. 엔드포인트, 서비스 로직, 데이터베이스 모델, 유틸리티 함수 등으로 구성되어 있습니다.
- `apps/imagene/ui`: React 기반의 프론트엔드 코드가 위치합니다. 컴포넌트, 페이지, 상태 관리, API 클라이언트 등으로 구성되어 있습니다.

## 💡 고급 기능 및 최적화

- **DNA 기반 생성**: 기존 이미지의 키워드 조합을 활용해 새로운 이미지를 생성하는 유전 알고리즘 적용
- **확률적 키워드 선택**: 키워드 사용 빈도를 기반으로 한 스마트한 프롬프트 생성
- **멀티 GPU 지원**: 여러 GPU를 활용한 병렬 이미지 생성으로 처리 속도 향상
- **벡터 임베딩 검색**: pgvector를 활용한 고성능 유사도 검색으로 이미지 분류 및 그룹화 자동화
- **성능 최적화**: 배치 처리, 메모리 관리, 데이터베이스 인덱싱 등 다양한 기법을 통해 시스템 성능 극대화
